Error codes (use these EXACT strings; openapi.yaml examples are authoritative):
- PKWT without end_date → `apperr.Invalid({"end_date":"..."})` → 400.
- end_date < start_date → `apperr.Invalid({"end_date":"..."})` → 400.
- PKWT (end_date - start_date) > 5 years → `apperr.Rule("PKWT_PERIOD_EXCEEDS_MAX", {"end_date":"Periode ... melebihi maksimum 5 tahun."})` → 422 (Rule() defaults 422).
- 2nd active agreement → `apperr.Conflict("ACTIVE_AGREEMENT_EXISTS")` → 409 (Conflict() forces 409; DO NOT use Rule — default would be 422).
- :renew on a non-active predecessor → `apperr.Conflict("CONFLICT")` → 409.
- :close on already closed/superseded → `apperr.Conflict("CONFLICT")` → 409.
- file > 10MB → `apperr.Error{Code:"FILE_TOO_LARGE", HTTPStatus: http.StatusRequestEntityTooLarge}` (413; struct literal because statusForCode has no 413 mapping — same technique as GEOFENCE_RADIUS_INVALID in Phase 3).
- disallowed mime → `apperr.Invalid({"file":"..."})` → 400.

OpenAPI Agreement schema (openapi.yaml 3035-3069), snake_case:
id, employee_id, type(PKWT|PKWTT), agreement_no, start_date, end_date(nullable),
status(ACTIVE|SUPERSEDED|CLOSED|EXPIRING), predecessor_id(nullable), successor_id(nullable),
closed_reason, closed_at(nullable), compensation{base_salary_idr,bpjs_terms{...4 pct fields},
tax_profile, effective_date}, created_at, updated_at.
- EXPIRING is a VIRTUAL status: compute at DTO boundary — if status='active' AND type='PKWT'
  AND end_date within 30 days of now → emit "EXPIRING" (persisted status stays 'active').
AgreementWriteRequest (3070): employee_id, type, agreement_no, start_date, end_date, compensation.
:renew body (1030-1050): type, agreement_no, start_date, end_date(req for PKWT), compensation, note.
:close body (1107-1123): reason(RESIGNED|TERMINATED|END_OF_TERM|OTHER), effective_date, note.

FileRef schema (openapi.yaml 2703): {id, url, name, size_bytes, mime, uploaded_at}.
Multipart form (1164-1181): field `file` (required, binary), `category`
(signed_agreement|addendum|supporting_doc, default signed_agreement), `caption` (≤200).
Max 10MB; allowed mime: application/pdf, image/jpeg, image/png.

Queries available (from 04-01 db/queries/people/agreements.sql + agreement_attachments.sql):
ListAgreements, GetAgreementByID, GetActiveAgreementForEmployee, CreateAgreement,
SetAgreementStatus, CreateAttachment, GetAttachmentByID.
