    - `func (c *Cipher) Encrypt(plaintext string) ([]byte, error)` — generate a random nonce (`c.aead.NonceSize()` bytes via crypto/rand), `out := c.aead.Seal(nonce, nonce, []byte(plaintext), nil)` (nonce PREPENDED to ciphertext), return out.
